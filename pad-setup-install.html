⚙️ インストールと事前準備

Power Automate for Desktop（PAD）を使い始める前に、
インストールと必要な設定を済ませておきましょう。Windows PC での手順を説明します。

1
Power Automate for Desktop をインストールする

Windows 11 には Power Automate for Desktop が標準搭載されています。
まずはスタートメニューから起動して、アプリケーションが存在するかを確認しましょう。

  1. スタートメニュー（左下の Windows ロゴ）をクリックする
  2. 検索バーに「Power Automate」と入力して検索する
  3. 「Power Automate」（Microsoft Corporation 製）が表示されたら、それをクリックして起動する
  4. Microsoft アカウント（職場・学校アカウント）でサインインする
💡 アプリが見つからない場合
検索しても「Power Automate」が見つからない場合は、以下の手順でインストールしてください：
Microsoft Store を開き、検索バーで「Power Automate」と入力し、
「Power Automate」（Microsoft Corporation 製）の「インストール」をクリックしてください。
💡 Microsoft アカウントについて
会社のメールアドレス（例：name@company.com）でサインインします。
個人の Microsoft アカウントではなく、職場・学校アカウントを使用してください。

まだ Microsoft アカウントを持っていない場合：
スタートメニュー → 設定 → アカウント から、職場または学校用アカウントの追加をクリックして新規作成できます。
2
Microsoft Edge の拡張機能をインストールする

PAD が Edge ブラウザを自動操作するために、専用の拡張機能をインストールする必要があります。
この設定をしないと、Web ページの操作が正しく動作しません。

以下のリンクから直接インストールするのが最も簡単です：

🔗 Microsoft Power Automate をインストール

上のリンクをクリックすると Microsoft Edge ストアが開き、「取得」ボタンで簡単にインストールできます。

または、手動でインストールすることもできます：

  1. Microsoft Edge を開く
  2. 右上の「…（設定など）」→「拡張機能」を選択
  3. 「Microsoft Edge アドオンを開く」をクリック
  4. 「Microsoft Power Automate」を検索してインストール
💡 インストール確認方法
Edge の右上にパズルピースのアイコンが表示され、一覧に「Microsoft Power Automate」が追加されていればOKです。
3
Edge のバックグラウンド実行をオフにする

Edge を閉じた後もバックグラウンドで動き続ける設定があります。
これがオンのままだと、PAD がフローの中で Edge を操作する際に予期しないエラーが発生することがあります。
必ずオフに設定しておきましょう。

以下の手順で設定を変更します：

  1. Microsoft Edge を開く
  2. 右上の「…（設定など）」をクリックし、「設定」を選択する
  3. 左側メニューの「システムとパフォーマンス」をクリックする
  4. 「システム」をクリックする
  5. 「Microsoft Edge が閉じられた後もバックグラウンドの拡張機能およびアプリの実行を続行する」を探す
  6. トグルスイッチを オフ（グレー）にする
⚙️ 設定 › システムとパフォーマンス › システム › バックグラウンド実行を続行する → オフ
オフ（グレー）の状態にしてください
⚠️ なぜオフにするの？
バックグラウンド実行がオンの場合、Edge の見えないプロセスが残ったままになります。
PAD がフローの中で「Edge を新しく開く」操作をした際に、古いプロセスと競合してエラーになることがあります。
🎉
準備完了！

3つの設定が完了したら、PAD を使う準備は万端です。
さっそくフローを作ってみましょう！

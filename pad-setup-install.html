⚙️ インストールと事前準備

Power Automate for Desktop（PAD）を使い始める前に、
インストールと必要な設定を済ませておきましょう。Windows PC での手順を説明します。

1
Power Automate for Desktop をインストールする

Windows 11 には Power Automate for Desktop が標準搭載されています。
まずはスタートメニューから起動して、アプリケーションが存在するかを確認しましょう。

  1. スタートメニュー（左下の Windows ロゴ）をクリックする
  2. 検索バーに「Power Automate」と入力して検索する
  3. 「Power Automate」（Microsoft Corporation 製）が表示されたら、それをクリックして起動する
  4. Microsoft アカウント（職場・学校アカウント）でサインインする
💡 アプリが見つからない場合
検索しても「Power Automate」が見つからない場合は、以下の手順でインストールしてください：
Microsoft Store を開き、検索バーで「Power Automate」と入力し、
「Power Automate」（Microsoft Corporation 製）の「インストール」をクリックしてください。
💡 Microsoft アカウントについて
会社のメールアドレス（例：name@company.com）でサインインします。
個人の Microsoft アカウントではなく、職場・学校アカウントを使用してください。

まだ Microsoft アカウントを持っていない場合：
スタートメニュー → 設定 → アカウント から、職場または学校用アカウントの追加をクリックして新規作成できます。
🎉
準備完了！

セットアップが完了したら、PAD を使う準備は万端です。
次は、ブラウザ操作のために WebDriver をインストールしましょう！

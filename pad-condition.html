🔀 条件分岐とは？ ― 「判断つきの自動化」の入口

「もし入社10年以上のスタッフならベテラン枠でリスト化、それ以外は通常リストに」――こうした判断を組み込めるのが IF / ELSE IF / ELSE アクションです。

① IF の基本構文

IF 勤続年数 >= 10 THEN ベテランリストに追加 ELSE IF 勤続年数 >= 5 THEN 中堅リストに追加 ELSE 新人リストに追加 END
PADでは「If」「Else if」「Else」の3つのアクションを組み合わせて使います。判定対象（変数）と演算子（=, >, <, >=, <=, <>）を選ぶだけのGUI操作です。

② インタラクティブデモ：スタッフ振り分けシミュレーター

入社日を入力して「判定する」を押すと、勤続年数に応じてベテラン／中堅／新人のいずれかに振り分けられます。

入社日を選んで「判定する」を押してください

③ 複数条件（AND / OR）

複数の条件を組み合わせるには AND（両方満たす）と OR（どちらかを満たす）を使います。

IF 部署 = "営業部" AND 役職 IN ["主任", "課長", "部長"] THEN 営業マネージャー候補リストに追加 END IF 部署 = "管理部" OR 部署 = "総務部" THEN バックオフィスリストに追加 END
注意： AND と OR を混在させる場合は、優先順位がわかりにくくなります。複雑な条件は IF を入れ子 にしたほうが読みやすくなることが多いです。

📝 3行まとめ

  1. IF / ELSE IF / ELSE で「判断つきの自動化」が組める。
  2. 演算子は = > >= など。条件の対象は変数や取得値。
  3. 複雑な条件は AND/OR の混在より、IF の入れ子のほうが読みやすい。
